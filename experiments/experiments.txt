
parameters for DiffusionMap:

    diffusion
        1. number of plans to sample (up_to)
        2. omega
        3. truncation dimension (dim)
        4. max cluster number (k_max)
        5. file ***optional***

    ensemble (ensemble, precinct, district, core, node)
        5. distance function (method)
            -> method = 'core', 'hamming'
        6. distance mode (mode)
            -> for method = 'core' -> mode = (power, 'normal'),
                                             (power, mu, sig),
                                             (power, mu_1, sig_1, mu_2, sig_2)
            -> for method = 'hamming' -> mode = 'equal', 'population', 'relative',
                                                'republican', 'democrat', 'voting_pop', 'vote' 
        7. forcing vector (data)
            -> data = 'equal', 'population', 'republican',
                      'democrat', 'voting_pop', 'vote'
    
    vertical
        8. value to be subtracted from core (subtract)
            -> subtract = 'min' or float
        9. type of matrix Q for Metropolis-Hastings (q_type)
            -> q_type = 'only_neighbours', 'all_precincts'        

    horizontal
        10. alpha for Gaussian kernel (alpha)
